# 场景方案数据库

## 使用说明
按「产品大类 → 问题类型 → 常规SOP → 超预期方案」结构组织。
超预期方案分三个层次：**产品层面**（解决问题本身做得更好）、**情感层面**（让用户感受到被重视）、**增值层面**（创造额外价值）。

---

## 🚨 零、售后政策红线（所有方案必须在此框架内）

> **红线是硬约束，任何超预期方案都不能踩线。价值创造 = 在红线内把方案做到最有温度。**

### 硬性规则

| 政策 | 规则 | 不可突破 |
|------|------|---------|
| **换货** | 只能换 **同款、同色、同型号** | ❌ 不能换不同款/不同色/升级款 |
| **退货** | 签收后 **7天内** 可退 | ❌ 超过7天不能走退货退款 |
| **换货次数** | 同一订单 **1-2年内只能换一次** | ❌ 不能反复换货 |
| **补发** | 质保内可补发同款 | ❌ 不能补发不同款 |

### ⭐ 红线内的价值创造思维

```
❌ 错误思路：超预期 = 突破政策给用户更多（换更好的款、额外补偿）
✅ 正确思路：超预期 = 在固定方案内改变「怎么做」和「怎么说」

方案是同一个（同款同色换货），但可以不一样的地方：
  → 方案节奏：先发后退 vs 先退后发（用户体验天差地别）
  → 服务细节：免运费、不催促、主动上门取件
  → 情感连接：把用户从"被处理的工单"变成"被理解的人"
  → 反馈闭环：帮用户记录问题→反馈产品组→后续跟进通知
  → 身份转变：从"抱怨者"变成"产品体验官"
```

### 红线检查清单（每次出方案前过一遍）

- [ ] 换货方案是否为同款同色同型号？
- [ ] 退货是否在签收7天内？
- [ ] 该订单是否已经换过货？
- [ ] 有没有承诺超出权限的补偿？

## 一、手机壳

### 1.1 壳体开裂/损坏
**常规SOP**：核实购买时间，在质保期内换新（⚠️ 同款同色同型号）。
**超预期方案**（红线内）：
- 【产品层面】先发后退（新壳先发出，用户收到后再寄回旧壳，手机全程有保护）
- 【情感层面】"壳裂了手机没事吧？这说明壳子替您扛了一下😊 先确认手机OK，我这边马上给您处理"→ 先关心手机安全再处理售后
- 【增值层面】记录用户反馈的问题→反馈产品组→告知用户"后续有改进我通知您"
- 【服务细节】运费我们出 + 不催寄回 + 用户方便时寄就行
- **话术**："亲，手机没磕到吧？壳子替您扛了一下😊 新的我先给您发出来，收到再把旧的寄回就行，运费我们出～"

### 1.2 壳体发黄/变色
**常规SOP**：告知属于正常氧化，非质量问题。
**超预期方案**：
- 【产品层面】"透明壳时间久了确实会氧化，我们新出的XX壳用的是新材料，抗黄变能力强很多，您可以了解一下"
- 【情感层面】"您用了这么久说明是真喜欢我们家产品呢～老用户我们是特别珍惜的"
- 【增值层面】如果是老用户，可以主动询问"您之前在我们家买过其他产品吗？"建立连接
- **话术**："亲，透明壳用久氧化是材料特性～我们新款抗黄材料升级了很多，下次换壳可以试试😊"

### 1.3 磁吸力不够（MagSafe壳）
**常规SOP**：确认手机型号兼容性，如不兼容告知，如兼容补发。
**超预期方案**：
- 【产品层面】帮用户排查：是壳的问题还是充电器的问题？教用户正确的对准位置
- 【情感层面】"磁吸这块确实需要对准位置，我教您一个小技巧"→ 主动教学
- 【增值层面】"如果您还没有MagSafe充电器的话，我们家有配套的，磁吸力会更强因为是配套设计的"
- **话术**："亲，您用的什么充电器？先试试对准中心位置找到'咔嗒'吸附感，还是弱的话拍个视频给我，我帮您判断要不要换新～"

## 二、手机膜

### 2.1 膜碎了/裂了
**常规SOP**：在保补发新膜。
**超预期方案**：
- 【产品层面】"膜碎了恰好说明它帮您挡了一次，屏幕肯定没事😊"
- 【情感层面】先确认屏幕安全，让用户安心
- 【增值层面】"新膜发给您的同时，我把贴膜教程也一起发您，上次可能贴的时候有小气泡影响了受力均匀"
- **话术**："亲，膜碎了说明替屏幕挡了一刀，屏幕肯定没事😊 我马上补发，贴膜攻略一起发您～"

### 2.2 贴膜有气泡/白边
**常规SOP**：指导用户重新贴或补发。
**超预期方案**：
- 【产品层面】发送图文/视频贴膜教程，手把手教
- 【情感层面】"贴膜确实是个技术活，不着急我一步步教您"→ 耐心指导
- 【增值层面】"如果实在贴不好也不用勉强，很多用户去线下店贴，我给您补发一张新的，让师傅帮您贴就行"
- **话术**："亲，用刮卡从中间往边缘推，90%气泡都能解决。实在不行我补发一张，去手机店让师傅帮贴就完美了～"

### 2.3 防窥膜影响显示效果
**常规SOP**：解释防窥膜特性，非质量问题。
**超预期方案**：
- 【产品层面】帮用户调整亮度设置，教最佳使用方法
- 【情感层面】"防窥膜确实需要适应一下，我帮您调到最佳效果"
- 【增值层面】"如果觉得防窥膜亮度太暗，要不要试试我们普通高清膜？我可以给您推荐"
- **话术**："亲，防窥膜会比普通膜暗一点，把亮度调到70%以上+开启原彩显示就好很多。还是不习惯的话我帮您看看其他方案～"

## 三、充电宝/移动电源

### 3.1 充不进电/无法充电
**常规SOP**：排查线材和接口，确认故障后换新。
**超预期方案**：
- 【产品层面】先远程排查（换线试试、检查接口有无异物、长按重启），避免用户来回寄
- 【情感层面】"充电宝突然没电确实很急人，特别是出门在外的时候。我先帮您快速排查一下"
- 【增值层面】"如果确实需要换新，您等新的到了再把旧的寄回就行，不影响您用"→ 先发后退
- **话术**："亲，先试三步：①换根线 ②清一下充电口 ③长按开关10秒重启。都不行我直接发新的，收到再寄回旧的😊"

### 3.2 充电慢
**常规SOP**：确认是否使用原装线材和匹配功率的充电头。
**超预期方案**：
- 【产品层面】帮用户计算：这款充电宝支持XX瓦，您手机支持XX瓦，需要用XX线才能达到最快速度
- 【情感层面】"很多用户不知道快充其实需要'三件套'都匹配才行，我帮您看看"
- 【增值层面】科普快充知识，顺便推荐匹配的充电线/充电器
- **话术**："亲，快充需要充电宝+线+手机三个都支持才能跑满速度。您用的什么线？普通线只能慢充，换快充线就好了～"

### 3.3 发热严重
**常规SOP**：告知边充边用会发热属正常。
**超预期方案**：
- 【产品层面】科普安全温度范围，教用户正确使用习惯
- 【情感层面】"发热我理解您的担心，安全最重要，我帮您判断一下"
- 【增值层面】"如果温度确实超出正常范围，安全起见我直接给您换新"
- **话术**："亲，40-50℃温热是正常的，但烫到拿不住就不正常了。您感受一下，异常的话我马上给您处理，安全第一😊"

## 四、充电器/充电头

### 4.1 无法快充/协议不兼容
**常规SOP**：确认手机支持的充电协议。
**超预期方案**：
- 【产品层面】帮用户查询手机支持什么充电协议，对比充电器支持的协议
- 【情感层面】"快充协议确实比较复杂，我帮您理清楚"
- 【增值层面】制作一张匹配表，告诉用户哪些产品搭配效果最好
- **话术**："亲，您手机是XX对吧？协议是兼容的，快充没生效很可能是线的问题——您用的什么线？有些线不支持快充传输～"

### 4.2 发热
**常规SOP**：GaN充电器高功率下微热正常。
**超预期方案**：
- 【产品层面】科普GaN充电器的散热原理，65℃以内正常
- 【情感层面】理解担心，给出明确安全标准
- 【增值层面】教用户延长充电器寿命的小习惯
- **话术**："亲，GaN充电器65℃以内都安全，温温的=正常，烫到拿不住=不正常。异常发烫请立即停用联系我😊 充电时放通风处别压被子下面～"

## 五、数据线

### 5.1 接口松动/接触不良
**常规SOP**：质保内换新。
**超预期方案**：
- 【产品层面】先排查是线的问题还是手机口的问题（换个线试）
- 【情感层面】"接口松这个确实影响使用体验，我来帮您判断"
- 【增值层面】教用户清理手机充电口（牙签轻轻掏出灰尘）
- **话术**："亲，先换根线试试：其他线也松=手机口积灰了，牙签掏一下就好；只有这根松=线的问题，我直接给您换新～"

### 5.2 断裂
**常规SOP**：质保内换新。
**超预期方案**：
- 【情感层面】"线断了如果是急用的话，我给您加急发"
- 【增值层面】教用户保护数据线的方法，推荐编织线款
- **话术**："亲，线断了我马上补发！拔线捏头别拽线能用更久，经常弯折的话下次试试编织线款更耐用😊"

## 六、散热器/散热背夹

### 6.1 噪音大
**常规SOP**：确认是否异常噪音，如是则换新。
**超预期方案**：
- 【产品层面】教用户调节档位，低档位更安静
- 【情感层面】"游戏的时候确实不想听到噪音，我帮您找到最佳平衡"
- **话术**："亲，试试调到中档，散热差别不大但安静很多。中档还是响的话我直接给您换一个～"

### 6.2 凝水问题
**常规SOP**：半导体制冷凝水正常。
**超预期方案**：
- 【产品层面】科普原理 + 教用户正确使用（不要在浴室等高湿环境用）
- 【情感层面】"凝水说明散热效果到位了，但确实要注意保护手机"
- **话术**："亲，凝水说明散热给力😊 注意避免潮湿环境使用，凝水了及时擦一下手机背面就好～"

## 七、通用超预期策略（适用所有品类）

### 7.1 用户情绪激动/不满
**超预期方案**：
- 不急着解决问题，先做情绪安抚
- 承认用户感受的合理性
- **话术**："亲，完全理解您的心情。您放心，问题一定帮您解决好，我先了解一下情况哈～"

### 7.2 用户是回头客/多次购买
**超预期方案**：
- 识别到老用户后主动表达感谢
- **话术**："亲，看到您买过好几次了，感谢老朋友的信任！您的问题我优先处理😊"

### 7.3 用户给家人/朋友买的
**超预期方案**：
- 顺势建立情感连接
- **话术**："亲，您帮[家人/朋友]买真的很贴心！使用中有任何问题随时找我们～"

### 7.4 用户提到要出差/旅行/急用
**超预期方案**：
- 加急发货 + 临时替代方案
- **话术**："亲，XX号出发是吧？我帮您加急发货争取赶上！万一来不及我给您一个临时方案～"

### 7.5 用户主动说"产品不错就是这个问题"
**超预期方案**：
- 这类用户最可能转化为传播者
- **话术**："亲，感谢认可！这个问题我帮您反馈给产品组，产品越来越好有您的功劳😊 这次我这样帮您处理——"

## ⭐ 八、人物画像侦测与差异化关怀（核心价值创造引擎）

> **核心理念**：方案是固定的（换货、退款、补发），谁来处理都一样。真正的价值创造 = **场景感知** — 从对话推敲出用户背后的生活场景，用现有方案的组合+人文关怀去回应。这是别人复制不了的，也是评审最看重的「创造性」。

### 使用方法
1. **扫描对话中的「场景线索」**（人物情境 + 行为场景）
2. **匹配到情境后，在常规方案基础上叠加关怀动作或方案组合**
3. **关怀是"加分项"，不替代解决问题本身**

### ⭐ 从案例 #001 学到的：行为场景侦测

> 不只看用户是谁（人物情境），还要看用户在做什么（行为场景）。

| 行为场景 | 侦测线索 | 创新方向 |
|---------|---------|---------|
| 🔄 多订单关联 | 同一用户短期多单/退换 | 跨订单串联，统一解决 |
| ⏳ 过渡保护 | 用户自行买替代品过渡 | 先发后退，减少垫资 |
| 🔁 重复进线 | 同一问题多次联系 | 升级方案，一次搞定 |
| 📦 拆分需求 | 一个问题分成多个操作 | 合并处理，简化流程 |

### 8.1 孕妈/准妈妈

**侦测信号**：
- 直接说："我怀孕了" "孕期" "待产"
- 间接线索："不方便出门" "去邮局不太方便" "弯腰不方便" "味道比较敏感"
- 收货地址有"妇幼""妇产"等关键词
- 购买了母婴相关搭配商品

**关怀动作**：
- 🚚 **主动安排上门取件**（退换货场景）："您现在不方便跑快递站的话，我帮您安排快递员上门取件，您不用出门"
- ⚡ **主动提出先发后退**："您现在特殊时期，新的我先给您发出，您什么时候方便再寄回旧的就行，不着急"
- 💡 **产品使用安全提醒**：充电宝→"孕期充电宝尽量不要放在肚子附近用哦"; 散热器→"这个散热器声音可以调小，晚上休息不受影响"
- 🎁 **温暖收尾**："祝您和宝宝都健健康康的😊"

**话术示例**：
"亲，问题我马上处理。退换的话我直接安排快递上门取件，您安心在家等就好😊"

**价值创造点**：从「产品售后」升级到「对人的关怀」，这种细节最打动评审。

---

### 8.2 老年用户/帮父母操作的子女

**侦测信号**：
- "我爸/我妈用的" "给老人买的" "父母不太会用"
- 直接聊天语气判断（打字慢、表达不太流畅、用语较正式）
- "不太懂这些" "我不太会弄"
- 购买简洁款/大字版等产品

**关怀动作**：
- 📝 **简化操作步骤**：把操作写成1234简单步骤，每步一行，不用术语
- 📸 **主动提供图片/视频教程**："我发给您一个图文教程，每一步都有截图，照着来就行"
- 👨‍👩‍👧 **如果是子女代问**："您放心忙，我把步骤写得很清楚，您直接转发给爸爸/妈妈就能看懂"
- 🔄 **安排子女代寄退**："如果老人不方便寄的话，等您周末回去的时候帮忙寄就行，不着急"
- 💡 **预判后续问题**：老人用充电器→"建议给叔叔/阿姨配一根带指示灯的线，充满了灯会变绿，一看就知道"

**话术示例**（子女代问场景）：
"亲，我帮您处理。步骤写简单点您截图发给爸爸就行：①长按按钮3秒 ②灯亮再插线 ③充2小时看看😊"

---

### 8.3 残障/行动不便用户

**侦测信号**：
- "我不方便出门" "行动不太方便" "轮椅" 
- 偶尔提到医院、康复、理疗等
- 语音转文字的特殊表达方式

**关怀动作**：
- 🚚 **100%安排上门取件**：不要让用户主动提
- ⏰ **时间上完全配合用户**："您什么时候方便我就让快递什么时候来"
- 📝 **减少用户操作**："您什么都不用准备，快递员来了直接交给他就行"
- ❤️ **自然不刻意**：不要过度表达同情，像朋友一样自然地帮忙

**话术示例**：
"亲，退回的事儿您不用管，我安排快递上门取，您说个方便的时间就行。新的先给您发出去😊"

**⚠️ 注意**：绝不说"您不方便""您身体不好"等暗示残障的话，要让关怀自然地融入流程。

---

### 8.4 出差/旅行/急用党

**侦测信号**：
- "要出差" "后天飞" "在外地" "旅游回来再说"
- "急用" "赶时间" "能不能快点"
- 收货地址是酒店
- 购买充电宝/充电器/数据线（旅行三件套）

**关怀动作**：
- ⚡ **时间线管理**："您是X号出发对吗？我帮您看下物流能不能赶上——如果发顺丰加急，大概X号能到"
- 🔧 **临时替代方案**："如果来不及的话，您可以先这样临时用：[临时方案]，回来了我再给您彻底处理"
- 📍 **灵活收货**："您在外地的话，新品可以直接寄到您出差的地址，退的等您回来再弄"
- 💡 **旅途产品建议**：充电宝问题→"出差的话建议带一个能上飞机的（<20000mAh），我看您这个可以的"

**话术示例**：
"亲，后天出差等不了了，我先帮您远程排查。解决不了的话：①今天顺丰加急 ②寄到您出差地址。您选哪个？"

### 8.5 送礼场景

**侦测信号**：
- "送人的" "给朋友买的" "当礼物"
- 收货地址和账号地址不同
- 购买了礼盒装/高端款
- "包装好看吗" "有没有袋子"

**关怀动作**：
- 🎁 **保护送礼体验**：退换货时注意不要让收礼人知道价格或问题
- 📦 **外包装处理**："如果需要补发的话，我备注一下让仓库包装得精致一点"
- ⏰ **配合时间节点**："您是什么时候送呀？我帮您卡好时间"
- 💡 **备选方案**："如果这个来不及换的话，同系列还有一款[更高端的]，直接下单明天就能到"

**话术示例**：
"亲，送朋友的我赶紧帮您处理。新的马上发出，备注仓库包装仔细些。什么时候要送？我帮您卡时间😊"

### 8.6 宝妈/带娃家庭

**侦测信号**：
- "带孩子不方便" "孩子把XX弄坏了" "小孩摔的"
- "孩子要用" "小朋友"
- 购买时间段在深夜（哄睡后的自由时间）

**关怀动作**：
- 👶 **安全第一提醒**："壳碎了没有碎片扎到小朋友吧？先确认宝宝安全哈"
- 🚚 **主动减少操作**：宝妈带娃很忙，退换流程越简化越好
- ⏰ **理解回复慢**：宝妈可能随时被打断，不要催
- 💡 **儿童防护建议**：充电器/数据线→"家里有小朋友的话，建议充完电把线收起来哦，小朋友容易拽着玩"

**话术示例**：
"亲，先看看宝宝有没有被碎片伤到，安全第一！壳子我直接给您发新的，小朋友调皮可以试试防摔等级最高的款😊"

### 8.7 情绪低谷/生活压力大的用户

**侦测信号**：
- 语气很丧："算了" "随便吧" "无所谓" "倒霉"
- "最近什么都不顺" "今天真倒霉"
- 产品问题只是压死骆驼的最后一根稻草
- 对很小的问题反应过大（情绪有积压）

**关怀动作**：
- ❤️ **不催不急不讲道理**：用户在情绪低谷时最不需要的就是"正常的哦""您看一下说明书"
- 🎯 **直接解决，减少用户参与**："您什么都不用管，我全部帮您处理好"
- 😊 **给一个小温暖**："希望我的处理能让您今天心情好一点点😊"
- ⚡ **特别高效**：低谷期的人受不了等待和反复沟通

**话术示例**：
"亲，交给我就好，您什么都不用管。我直接安排[方案]，到货就能用。希望能让您心情好一点😊"

### 8.8 学生党/预算敏感用户

**侦测信号**：
- 收货地址有"学校""大学""宿舍""学院"
- "学生" "没什么钱" "便宜点"
- 选购时反复比价、纠结
- 购买的是入门款/性价比款

**关怀动作**：
- 💰 **关注经济成本**：换而不退（避免用户等退款的资金占用期）
- 🚚 **减免邮费**："运费我们这边承担，您不用出"
- 💡 **使用技巧延长寿命**：学生换新成本高，多教产品保养方法
- 📝 **简洁高效**：学生课多时间紧，不啰嗦

**话术示例**：
"亲，直接帮您换新，运费我们出。教您个小方法：[产品保养技巧]，能用更久😊"

### 8.9 情境组合侦测——高价值场景

> 当一个用户身上同时叠加多个情境标签时，价值创造空间最大。

| 组合情境 | 侦测示例 | 关怀策略 |
|---------|---------|---------|
| 孕妈 + 急用 | "孕晚期待产包要用" | 上门取件 + 加急发货 + 祝福 |
| 老人 + 情绪低谷 | "我爸用了半天弄不好，急得不行" | 超简步骤 + 视频教程 + 安慰子女 |
| 送礼 + 时间紧 | "后天生日来不及了" | 顺丰加急 + 备选方案 + 包装备注 |
| 学生 + 产品损坏 | "在宿舍摔了，刚买没多久" | 快速换新 + 免运费 + 保护技巧 |
| 宝妈 + 愤怒 | "孩子差点被碎片伤到！" | 先问孩子安全 + 道歉 + 立即处理 + 安全建议 |
| 出差 + 多产品问题 | "出差路上充电宝和线都坏了" | 一站式解决 + 打包发顺丰 + 旅途建议 |

**处理原则**：
1. **先处理情绪最强的那个信号**
2. **关怀动作叠加，但话术不要太长**——最多选2-3个关怀点融入对话
3. **自然融入，绝不刻意**——说"我帮您安排上门取"比"考虑到您怀孕不方便出门"更自然

### 8.10 人物情境侦测自检清单

> 每次接待用户时，快速过一遍这个清单：

| 维度 | 问自己 | 线索来源 |
|------|--------|---------|
| 身体状况 | 是否有行动不便/孕期/残障的线索？ | 用户原话、地址 |
| 家庭角色 | 是给谁用的？帮谁买的？有小孩？ | 对话内容、订单信息 |
| 时间压力 | 是否急用？有时间节点？ | 用户原话 |
| 经济状况 | 是否对费用敏感？学生？ | 地址、选购行为 |
| 情绪背景 | 产品问题是不是只是最后一根稻草？ | 语气、用词 |
| 使用场景 | 在哪里用？什么情况下用？ | 对话推敲 |

**黄金法则**：方案解决的是「产品问题」，关怀解决的是「人的问题」。方案大家都一样，关怀才是你和别人的区别。
