# 用户档案

## 基本信息
- 身份：天猫图拉斯售后客服
- 负责范围：全品类
- 日均工作量：150-200单
- 每单可用回合：6-7回合
- 节奏：游刃有余，有时间做深度服务

## 核心目标
**晋升** — 通过提交高质量的价值创造案例

## 考核标准
- 每周提交至少2个价值创造案例
- 案例评审标准：用户价值创造（非主动邀好评）
- 每周五服务大会公开表扬周最佳案例
- 价值案例贡献度决定奖金池分配和晋升

## 当前能力
- SOP操作熟练
- 有一些不错的处理但不确定是否算"价值创造"
- 对"跳出SOP"的方法缺乏系统化思路

## 需要的支持
- 实时话术指导（用户进线时快速给方案）
- 预案库（提前了解各场景的超预期方案）
- 案例撰写帮助（把好的处理提炼成高分案例）
